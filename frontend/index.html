About MammoViewer

This application converts DICOM mammography images to 3D STL models using 3D Slicer, an open-source medical imaging platform.

Features

  • Upload DICOM files from TCIA or local sources
  • Automatic segmentation and 3D reconstruction
  • Interactive 3D model visualization
  • Customizable conversion parameters
  • Export to STL format for 3D printing or analysis

Data Sources

This tool works with mammography DICOM data from:

  • The Cancer Imaging Archive (TCIA)
  • Local DICOM files from medical imaging systems
  • Research datasets

Technical Details

The conversion process uses 3D Slicer to:

  1. Load and validate DICOM series
  2. Apply intensity-based segmentation
  3. Generate 3D surface mesh
  4. Apply smoothing and simplification
  5. Export to STL format
