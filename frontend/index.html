About MammoViewer

Features

  • Convert mammography DICOM files to 3D printable STL models
  • Adjustable threshold, smoothing, and decimation parameters
  • Real-time 3D preview with Three.js
  • Batch processing support

Data Sources

Test with public mammography datasets from:
The Cancer Imaging Archive (TCIA)

Technical Details

  • Backend: Python, Flask, SimpleITK
  • 3D Processing: 3D Slicer
  • Frontend: HTML5, CSS3, JavaScript, Three.js
  • Supported formats: .dcm, .dicom
